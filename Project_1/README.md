## Running a dedicated server on AWS EC2 platform based on Linux.


| Parameter                  |                         |
| ---------------------------|:-----------------------:|
| Amazon Machine Image       | Ubuntu Server 20.04 LTS |
| Instance Type              | t2.micro                | 
| Configure Instance Details | default                 |
| Add Storage                | default                 |
| Add Tags                   | default                 |
| Configure Security Group   | default                 |
